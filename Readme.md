This project is a portable temperature and humidity sensor made with Arduino.

Parts I used:
  - Elegoo Arduino R3
  - 220 Ω Resistors
  - 10k Ω Rotary Potentiometer (linear)
  - Arduino Liquid Crystal Display 16 x 2
  - DHT Humidity and Temperature Sensor
  - Duracell 9V battery
  - Arduino Breadboard
  - Jumper Wires
